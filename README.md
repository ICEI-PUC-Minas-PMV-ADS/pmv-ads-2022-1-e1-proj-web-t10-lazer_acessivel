# LAZER ACESSÍVEL 

`Curso: Análise e Desenvolvimento de Sistemas`

`Disciplina: Projeto: Desenvolvimento de Aplicação Web Front-End`

`1º semestre`

O lazer acessível é um site para buscar por estabelecimentos ou espaços públicos acessíveis e voltados para o lazer que podem ser acessados e utilizados por toda a população.
